# Dynamic Quiz Application

This project is a simple dynamic quiz application developed as part of a placement assessment.  
The application is built using basic **HTML, CSS, and JavaScript** and runs completely on the client side without using any backend or database.

The main focus of this project is to demonstrate logical thinking, JavaScript fundamentals, and UI automation readiness.

---

## Features

- Displays one question at a time
- Each question has 4 options labeled as A, B, C, and D
- 10 questions in total
- 10-second timer for each question
- Automatically moves to the next question when time expires
- Score is calculated using JavaScript
- Final result shows total score, correct answers, and wrong answers
- Simple and responsive layout

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- Selenium WebDriver (for automation testing)
